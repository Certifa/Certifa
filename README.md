# Hey, I'm Mike 👋

Cybersecurity & Cloud student at **Hogeschool Utrecht**, with a focus on offensive security. I enjoy breaking into things (legally), building tools that make pentesting workflows smoother, and documenting what I learn along the way.

When I'm not in a lab environment, you'll find me solving HTB machines, writing Python scripts, or diving into Active Directory internals.
